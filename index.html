ACS Web Inspector

Microsoft Office Agent の ACS をブラウザ内で解析し、画像・音声・アニメーションを抽出します。
2.0 系の flat ACS を主対象にしています。1.5 系の OLE/CFB はストリーム一覧と生データ抽出を行います。
